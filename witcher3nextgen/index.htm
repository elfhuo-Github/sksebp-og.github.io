SETTINGS FILE

Only needed if you want to install more/different mods than those listed.

Open "mods.settings" with a notepad.
*File location: "C:\Users\(*Username*)\Documents\Witcher 3\".
This file includes all mods, with two tags under each. One that enables/disables the mod, and one that decides priority.
The file already includes everything listed in this guide with the right priority for each mod.

If you installed a different version of a mod, then all you need to change is that mod's name. *Open the "mods" folder inside the Witcher 3 game folder to see all mod names. Find the one in question and remember it's folder name. Then find the same mod in the mods.settings file, and edit it to be the exact same name as it had in the mods folder.
If you didn't install all the listed mods, then you need to remove the entries of those mods from the file. The priorities of the rest don't need to be changed.
If you installed more mods than listed in this guide, then you need to add those mods to the file, keeping the same format, and setting the right priority.

Info: Witcher 3 handles mods in a completely different way than e.g. the Elder Scrolls games. Witcher 3 loads all mods into memory, but identical data is not overwritten. If two mods try to use the same data at the same time (conflict), the mod with the lowest number gets priority. (Completely backwards, if you are used to modding Skyrim)
The Witcher 3 has 9999 number sluts. Low numbers take priority to high numbers. If you want the content of mod X to overrule the content of mod Y, then mod X has to be assigned the lowest number of the two, which equals the highest priority.
Here is the tricky part: The game also uses this list as a load-order. It loads from top to bottom, so if you install a mod that depends on another mod, then that mod needs to be loaded before the one with the dependency.

GAME INFO

The Witcher 3: Wild Hunt is a critically acclaimed action role-playing game developed by CD Projekt Red and released in 2015 for PC, Xbox One, and PlayStation 4. The game, based on the series of fantasy novels by Polish author Andrzej Sapkowski, features an open-world environment, a complex branching narrative, and a deep combat system.

The story of The Witcher 3 follows Geralt of Rivia, a monster hunter known as a witcher, as he searches for his adopted daughter Ciri, who has been hunted by the Wild Hunt, a group of spectral riders. Players must make choices throughout the game that affect the outcome of the story and the fate of the characters. The game features a vast open-world environment, the Northern Kingdoms, filled with a variety of locations to explore, from dense forests to bustling cities.

The gameplay of The Witcher 3 is built around a deep combat system that allows players to use a variety of weapons, spells, and tactics to take down enemies. Players can also use the game's alchemy system to create potions and oils that can be used to enhance their abilities. The game's skill tree allows players to customize their playstyle, whether they prefer to focus on swordplay, magic, or stealth.

The game's characters are well-written and well-acted, with many of them having their own unique storylines and personalities. Players can also engage in romances with various characters, including Triss Merigold, Yennefer of Vengerberg, and Keira Metz.

The graphics of The Witcher 3 are absolutely stunning, with detailed character models, lush environments, and dynamic weather effects. The game's next-gen update, released for PlayStation 5 and Xbox Series X/S, improves on the already impressive graphics, offering even more detailed character models, higher resolution textures and improved lighting.

The development of The Witcher 3 was a long and complex process, with the game going through several iterations before its final release. The game's developer, CD Projekt Red, is a Polish studio known for their attention to detail and commitment to quality. The game's next-gen update was also developed with care, taking into account all the feedback from the players and the latest technology advancements.

Overall, The Witcher 3: Wild Hunt is a game that offers a truly unique and immersive experience. With its complex narrative, deep combat system, and stunning graphics, it is a game that will keep players engaged for hours on end. The next-gen update takes an already fantastic game and brings it to new heights, offering an even more immersive experience with the latest technology. If you're a fan of action role-playing games and fantasy worlds, The Witcher 3 is a must-play.
